# Breakeven Analysis – Idea‑Forge

| Item | Value | Notes |
|------|-------|-------|
| **Variable cost per active user / month** | **$0.025** |  • Compute: $0.010 <br>  • Storage: $0.005 <br>  • Bandwidth: $0.010 |
| **Pricing tiers** | | |
| • Basic | **$5 / mo** | Core ideation engine, 10 idea prompts / month, community forum |
| • Pro | **$15 / mo** | Unlimited prompts, idea validation checklist, export to CSV |
| • Enterprise | **$30 / mo** | All Pro features + API access, priority support, custom integrations |
| **Customer Acquisition Cost (CAC)** | **$200 – $400** | Based on paid ads + community outreach |
| **Lifetime Value (LTV)** | | |
| • Basic | **$48** | 12 mo × $5 × 80 % gross margin |
| • Pro | **$144** | 12 mo × $15 × 80 % |
| • Enterprise | **$288** | 12 mo × $30 × 80 % |
| **Break‑even users (acquisition‑only)** | **≈ 3 users** |  CAC / (LTV – 12 mo variable cost) ≈ 300 / (144 – 0.30) ≈ 2.1 → round up |
| **Path to $10 k MRR** | | |
| • Basic | **2,000 users** | 2,000 × $5 = $10 k |
| • Pro | **667 users** | 667 × $15 ≈ $10 k |
| • Enterprise | **334 users** | 334 × $30 ≈ $10 k |

> **Bottom line** – With a variable cost of only $0.025/user/month and a gross margin of 80 %, Idea‑Forge is profitable from the first dollar of revenue. The only real hurdle is acquiring the first 3 paying users; after that, the business scales linearly with user growth.
