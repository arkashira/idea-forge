# IdeaForge Onboarding
A simple onboarding flow for IdeaForge.

## Features
- 3-step onboarding wizard: interests, preferences, and first idea generation
- User preferences persisted across sessions
- Tooltips guide users through key features after first use

## Usage
1. Run `python -m pytest` to test the onboarding wizard.
2. Use the `OnboardingWizard` class to create a new onboarding flow.
